# 出前どれ得？

Uber Eats・出前館・Wolt・menu・公式アプリ/持ち帰りの中から、今日どのサービスから確認すると判断しやすいかを提案する6問診断です。

## 診断内容

- 6問の回答から5つの確認候補を簡易スコアリング
- 結果は価格順位ではなく「おすすめ確認順」として表示
- 公式アプリ・公式デリバリー・持ち帰りも候補に含む
- 8種類の結果カードを用意

## 技術

- Next.js
- React
- TypeScript

## ローカル開発

```bash
npm install
npm run dev
npm run build
```

## 免責事項

本サービスはリアルタイム価格比較や最安保証を行うものではありません。スクレイピング・API連携・注文代行も行いません。

実際の価格・配送料・手数料・クーポン適用条件は、地域・店舗・時間帯・アカウント状況によって変わります。最終的な金額は各アプリの注文確認画面で確認してください。
